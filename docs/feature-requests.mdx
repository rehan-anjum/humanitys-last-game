## Bugs

If something is broken, please open an issue at [github.com/vats98754/humanitys-last-game/issues](https://github.com/vats98754/humanitys-last-game/issues) including:

- HLG version (`python -c "import hlg; print(hlg.__version__)"`)
- Python version
- Minimal reproducible example (please include a level id and the action sequence that triggers the issue)
- Expected vs observed behavior

## Feature requests

The current roadmap is documented on [Changelog](/changelog) under the `Planned` heading. If your request is on it, comment on the existing issue with your use case. If not, open a new issue.

## Security disclosures

For security issues, contact the maintainer directly rather than opening a public issue.
