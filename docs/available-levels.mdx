HLG ships 34 hand-curated Bloxorz levels organized into four subsets. The semi-private set is used for the official frontier-model leaderboard; the fully-private set is reserved for the HLG Challenge.

## Discovering levels programmatically

```python theme={null}
import hlg
arc = hlg.Arcade()
for info in arc.list_levels():
    print(info.level_id, info.optimal_solution_length, info.has_switches)
```
