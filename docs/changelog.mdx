## v0.1.0 (current)

**Initial public release.**

- 34 hand-curated Bloxorz levels in `levels/`.
- Pure-Python engine: state, validator, BFS dead-state solver in `engine/`.
- Verified-optimal solutions for every level in `verify_solutions.py`.
- `hlg.Arcade()` Python SDK with `OperationMode.OFFLINE` (~150 LOC).
- Eight-event JSONL logging schema (`agent_logger.py`).
- HLG-RHAE scoring metric (`hlg.scoring`, `scripts/compute_rhae.py`).
- `extract_leaderboard.py` aggregator over `logs/`.
- Mintlify documentation site (this site).
- LaTeX research paper (`paper/`).
- OpenAPI 3.1 spec for the planned REST surface (`docs/hlg-openapi.yaml`).
