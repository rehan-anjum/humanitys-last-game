## Local

Run levels locally via the in-process engine. Recommended for development, iteration, and any agent that does not need shared scorecard infrastructure.

```python theme={null}
from hlg import Arcade, OperationMode

arc = Arcade(operation_mode=OperationMode.OFFLINE)
env = arc.make("level0")
```

`OperationMode.OFFLINE` is the default in v0.1 and the only mode currently supported.

## Online

`OperationMode.ONLINE` is reserved for HLG v0.2. It will let you:

- Submit runs to the official leaderboard.
- View interactive replays via shareable URLs.
- Use a hosted endpoint without managing the Python engine yourself.

The REST surface is published in v0.1 as `docs/hlg-openapi.yaml` ([Overview](/rest_overview)) so that integrators can prepare ahead of v0.2.
