## How this leaderboard is built

1. `python main.py --models <model> --level <ids>` runs the harness and writes
   per-(model, level) JSONL traces into `logs/`.
2. `python scripts/extract_leaderboard.py` walks every run, summarises tokens
   and first-win action counts, and writes `data/leaderboard.json`.
3. `python scripts/compute_rhae.py --print-summary` prints the table above
   in plain text. This page is regenerated from the same JSON.
