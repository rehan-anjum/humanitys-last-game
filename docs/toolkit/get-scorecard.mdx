```python theme={null}
import hlg

arc = hlg.Arcade()

# Default scorecard (created lazily on first make()).
sc = arc.get_scorecard()

# Or fetch by id.
sc = arc.get_scorecard(scorecard_id="abc-123-...")
```

## What's in a scorecard

```python theme={null}
sc.scorecard_id           # str
sc.tags                   # list[str]
sc.source_url             # str | None
sc.opaque                 # dict | None
sc.created_at             # ISO 8601
sc.closed                 # bool
sc.levels                 # {level_id: [LevelResult, ...]}
sc.first_win_actions(lid) # int | None — actions on first WIN of `lid`
sc.score                  # HLG-RHAE
```

## Persisting to disk

```python theme={null}
sc.to_jsonl("scorecards/my-run.jsonl")
```

Each line is one `LevelResult`:

```json theme={null}
{"level_id": "level5", "attempt_num": 1, "state": "WIN", "actions": 47, "timestamp": "..."}
```

This format is consumable by `scripts/extract_leaderboard.py` and the official analysis tooling.
