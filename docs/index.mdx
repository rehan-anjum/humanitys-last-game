Traditionally, AI is measured with static benchmarks — fine for evaluating LLMs on language and reasoning chains, but insufficient for evaluating frontier **agentic** systems. HLG measures four capabilities the static-benchmark format cannot capture:

- **Exploration** — actively discovering bridge structure and weak-tile placement
- **Causal Modeling** — predicting persistent side-effects of switch activations
- **Goal Decomposition** — identifying intermediate sub-goals (open bridge X before crossing)
- **Long-Horizon Planning** — sequencing 100+ moves where greedy proximity to the goal is wrong

By building agents that play HLG, you contribute directly to AGI evaluation research. [Learn more about HLG](/hlg).

## Play your first HLG level

### 1. Install the HLG toolkit

```bash theme={null}
uv init
uv add hlg
# or pip install hlg
```

### 2. Set your `HLG_API_KEY` (optional)

The default `OperationMode.OFFLINE` runs entirely locally and needs no key. An API key is only required for `OperationMode.ONLINE` (planned for v0.2) which enables remote scorecards and replays.

```bash theme={null}
export HLG_API_KEY="your-api-key-here"
# or
echo 'HLG_API_KEY=your-api-key-here' > .env
```

### 3. Play your first level

Create a file called `my-play.py`:

```python theme={null}
import hlg
from hlg import GameAction

arc = hlg.Arcade()
env = arc.make("level0", render_mode="terminal")

obs = env.reset()
for _ in range(10):
    obs = env.step(GameAction.RIGHT)
    if obs.state != hlg.GameState.NOT_FINISHED:
        break

print(arc.get_scorecard().score)
```

Run it:

```bash theme={null}
uv run python my-play.py
```

You should see the level rendered in your terminal and an HLG-RHAE score.
