## Planned (v0.2)

```python theme={null}
import os
import hlg
from hlg.providers.anthropic import AnthropicAgent

arc = hlg.Arcade()

agent = AnthropicAgent(
    model="claude-opus-4-6",
    api_key=os.environ["ANTHROPIC_API_KEY"],
    reasoning_effort="high",
    system_prompt_path="system_prompt.md",
)

env = arc.make("level5", render_mode="terminal")
agent.play(env)

print(arc.get_scorecard().score)
```

## Today (v0.1) — bring your own loop

You can build the same thing today using the official `anthropic` SDK. Read the canonical system prompt from `system_prompt.md`, build the per-turn user message in the format described on [LLM Agents](/llm_agents), and call `Anthropic().messages.create(...)` per turn.

The full v0.1-compatible example is documented in [Create an Agent](/create-agent#llm-driven-policy).
