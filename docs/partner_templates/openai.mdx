## Planned (v0.2)

```python theme={null}
import os
import hlg
from hlg.providers.openai import OpenAIAgent

arc = hlg.Arcade()

agent = OpenAIAgent(
    model="gpt-5.4",
    api_key=os.environ["OPENAI_API_KEY"],
    reasoning_effort="high",
    system_prompt_path="system_prompt.md",
)

env = arc.make("level5")
agent.play(env)
```

## Today (v0.1)

Use the official `openai` SDK directly. The canonical pattern (system prompt + per-turn user message + single-character response) is documented on [Create an Agent](/create-agent).
