## Planned (v0.2)

```python theme={null}
import os
import hlg
from hlg.providers.huggingface import HuggingFaceAgent

arc = hlg.Arcade()
agent = HuggingFaceAgent(
    model="meta-llama/Meta-Llama-3.1-70B-Instruct",
    api_token=os.environ["HF_TOKEN"],
)
env = arc.make("level0")
agent.play(env)
```
